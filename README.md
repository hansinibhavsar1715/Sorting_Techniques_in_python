## 📊 Sorting Algorithms in Python

This repository is a collection of popular sorting algorithms implemented in Python. It is created to help understand the core logic behind different sorting techniques in a simple and practical way.

Sorting is a fundamental concept in computer science, and learning it properly builds a strong base for data structures, problem-solving, and technical interviews.

## 🚀 Algorithms Included

* 🔹 Bubble Sort
* 🔹 Selection Sort
* 🔹 Insertion Sort
* 🔹 Merge Sort
* 🔹 Quick Sort
* 🔹 Counting Sort
* 🔹 Heap Sort
* 🔹 Tree Sort
* 🔹 Shell Sort
* 🔹 Radix Sort
* 🔹 Bucket Sort

## 🧠 What You Will Learn

* How different sorting algorithms work
* Comparison between sorting techniques
* Time and space complexity basics
* When to use which sorting algorithm
  
## ⏱️ Time & Space Complexity

| Algorithm      | Time Complexity (Best) | Average Case | Worst Case | Space Complexity |
| -------------- | ---------------------- | ------------ | ---------- | ---------------- |
| Bubble Sort    | O(n)                   | O(n²)        | O(n²)      | O(1)             |
| Selection Sort | O(n²)                  | O(n²)        | O(n²)      | O(1)             |
| Insertion Sort | O(n)                   | O(n²)        | O(n²)      | O(1)             |
| Merge Sort     | O(n log n)             | O(n log n)   | O(n log n) | O(n)             |
| Quick Sort     | O(n log n)             | O(n log n)   | O(n²)      | O(log n)         |
| Counting Sort  | O(n + k)               | O(n + k)     | O(n + k)   | O(k)             |
| Heap Sort      | O(n log n)             | O(n log n)   | O(n log n) | O(1)             |
| Tree Sort      | O(n log n)             | O(n log n)   | O(n²)      | O(n)             |
| Shell Sort     | O(n log n)             |    -         | O(n²)      | O(1)             |
| Radix Sort     | O(nk)                  | O(nk)        | O(nk)      | O(n + k)         |
| Bucket Sort    | O(n + k)               | O(n + k)     | O(n²)      | O(n)             |
